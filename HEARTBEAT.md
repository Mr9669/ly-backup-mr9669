# HEARTBEAT.md - 最轻量心跳（省 Token 版）

## 设计原则
- 只做必要的检查，不做多余操作
- 优先读文件元信息，不读内容
- 无异常不汇报，汇报只说关键数据

## 每次心跳执行
1. **快速检查**：ls -lt 检查最新文件时间戳（不读内容）
2. **异常检测**：对比时间戳，判断任务是否正常
3. **静默模式**：无异常 → 回复 HEARTBEAT_OK

## 判断规则
- 纵地金光：上次采集 > 14 小时 → 异常
- 星舰回测：今日 03:00 未运行 → 异常
- 迁移包：> 7 天未更新 → 警告

## 汇报模板（仅异常时使用）
```
⚠️ 异常:
- [任务]: [问题描述]
- 上次运行: [时间]
- 建议: [操作]
```

## 自动任务（独立运行，不依赖心跳）
- 纵地金光：每12小时 cron 调度
- 星舰回测：每日 03:00 cron 调度
- 备份：每日 04:00 cron 调度
