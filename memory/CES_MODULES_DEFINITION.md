### 2.2 关系说明

**钧天与陆压：**
- 钧天是陆压的"记忆宫殿"，存储所有知识和数据
- 陆压通过钧天获取信息，通过分析产生智慧
- 钧天是陆压进化的数据基础

**纵地金光与钧天：**
- 纵地金光是钧天的"输入管道"
- 采集的数据经处理后存入钧天
- 形成"采集→存储→分析→应用"的闭环

**陆压与模块层：**
- 陆压通过108工具与可插拔模块交互
- 各模块可以调用陆压的智能能力
- 陆压通过钧天监控各模块的运行状态
